# wikipedia-search-application
A responsive Wikipedia Search App built using HTML, CSS, Bootstrap, and JavaScript. It fetches real-time search results using the Fetch API and dynamically displays titles, links, and descriptions with a loading spinner for better user experience

## 🚀 Features
- Search articles by keyword
- Fetch real-time data using Fetch API
- Dynamic DOM manipulation
- Loading spinner while fetching data
- Responsive design using Bootstrap

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 4
- JavaScript (ES6)
- Fetch API
